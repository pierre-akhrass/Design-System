# Feedback

A notification component for displaying inline alerts and toast messages. Supports two variants: **alert** (light) and **toast** (dark).

---

## Overview

The Feedback component is used to communicate status messages, warnings, or informational notices to users. It supports:

- Two visual variants: `alert` and `toast`
- Dismissible with close button
- Info icon indicator
- Title and description content

---

## Variants

### Alert

Light background with subtle border:
- White background (`--sds-color-background-default-default`)
- Border: `--sds-color-border-brand-default` (`#BCBCBC`)
- Dark text for title and description

### Toast

Dark background for prominent notifications:
- Background: `--sds-color-background-default-secondary-system-primary` (`#141F2E`)
- Border: `--sds-color-border-brand-secondary` (`#6B6B6B`)
- White text (`--Schema-Text-On-Primary`)
- Box shadow for elevation

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'alert' \| 'toast'` | `'alert'` | Display variant. |
| `title` | `string` | — | **Required.** Title text. |
| `description` | `string` | — | **Required.** Description text. |
| `dismissible` | `boolean` | `true` | Show/hide close button. |
| `onDismiss` | `() => void` | — | Callback when dismissed. |
| `className` | `string` | — | Additional CSS class names. |

## Design Tokens

### Colors (CSS Custom Properties)

```css
/* Alert variant */
--sds-color-background-default-default: /*slate surface*/; /* Alert background */
```

## Accessibility

- Container uses `role="alert"` for screen reader announcement
- Close button has `aria-label="Dismiss"`
- Decorative icons use `aria-hidden="true"`

---

## Usage

```tsx
import { Feedback } from '@/components/Feedback'

// Alert variant (default)
<Feedback
  title="This is the title"
  description="A problem was encountered while processing your request."
/>

// Toast variant
<Feedback
  variant="toast"
  title="This is the title"
  description="A problem was encountered while processing your request."
/>

// Non-dismissible
<Feedback
  title="Info"
  description="This cannot be closed."
  dismissible={false}
/>
```
