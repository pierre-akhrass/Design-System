# Quote

A carousel-based quote/testimonial component that displays customer quotes with navigation, author information, and dot indicators. Supports two layout variants: **default** (centered, full-width) and **inline** (card-based).

---

## Overview

The Quote component is designed for displaying customer testimonials or quotes in a carousel format. It supports:

- Multiple quotes with carousel navigation
- Prev/Next navigation buttons with author avatar
- Dot indicators for direct navigation
- Two layout variants: `default` and `inline`
- Responsive design (mobile-optimized)

---

## Variants

### Default

Centered layout with:
- Quote icon (52├Ś38px, `#545454`)
- Italic quote text in curly quotes (`"..."`)
- Prev/Next navigation with centered avatar (120├Ś120px)
- Author name and role below avatar
- Dot indicators at the bottom

### Inline

Card-based layout with:
- Background card (`--sds-color-background-default-tertiary-system` / `#D2D9E0`)
- Left-aligned content
- Quote icon at the top
- Italic quote text (no curly quotes)
- Author row: small avatar (64├Ś64px) + name/role
- Dot indicators below the card

## Responsive Behavior

### Mobile (Ōēż 768px)

- **Default variant:** Quote text reduces to `24px`; author name/role is hidden (avatar still visible in nav)
- **Inline variant:** Card padding reduces to `32px 24px`

---

## Accessibility

- Navigation buttons have `aria-label` attributes (`"Previous quote"`, `"Next quote"`)
- Dot indicators have `aria-label` (`"Go to quote {n}"`)
- Decorative SVGs use `aria-hidden="true"`
- Image placeholders use `aria-hidden="true"`
- Semantic `<blockquote>` element for quote text

## Usage

```tsx
import { Quote } from '@/components/Quote'

const quotes = [
  {
    image: '/avatars/user1.jpg',
    quote: 'An amazing experience that exceeded all expectations.',
    name: 'Jane Doe',
    role: 'Customer',
  },
  {
    image: '/avatars/user2.jpg',
    quote: 'Outstanding service and quality throughout.',
    name: 'John Smith',
    role: 'Partner',
  },
]

// Default variant
<Quote quotes={quotes} />

// Inline variant (card style)
<Quote quotes={quotes} variant="inline" />

// Without navigation
<Quote quotes={quotes} showNavigation={false} />

// Without author info
<Quote quotes={quotes} showAuthor={false} />
```
